## Project Overview
This project analyzes sales transaction data to identify top-performing product categories and revenue trends. It simulates a real-world retail dataset to demonstrate data cleaning, aggregation, and visualization techniques using Python.

## Key Features
- **Automated Data Generation:** Script creates a realistic dataset of 1,000 transactions.
- **Exploratory Data Analysis (EDA):** Calculates total revenue, average order value (AOV), and category performance.
- **Visualization:** Generates a bar chart comparing sales performance across different sectors.

## Technologies Used
- **Python 3.x**
- **Pandas** (Data Manipulation)
- **Matplotlib & Seaborn** (Data Visualization)
- **NumPy** (Numerical Operations)

## Results
- Identified **Electronics** as the highest revenue-generating category.
- Visualized sales distribution to aid in inventory planning.
- *See `sales_chart.png` for the visual breakdown.*

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install pandas numpy matplotlib seaborn`
3. Run the script: `python sales_analysis.py`
